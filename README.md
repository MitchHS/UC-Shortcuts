# UC-Shortcuts

*Simple android app that provide useful links to University of Canberra Students*

1. UC Map - Available offline
2. UC Email
3. UC Allocate
4. UC Canvas
5. UC Exam TimeTable
6. UC Home 

Should be support by most mobile resolutions and a majority of tablets. Work in progress.
